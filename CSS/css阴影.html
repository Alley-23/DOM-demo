CSS投影

box-shadow: 右移，下移，模糊，扩张半径，颜色；

模糊：使用高斯模糊算法(或类似算法)将它进行4px的模糊处理。这在 本质上表示在阴影边缘发生阴影色和纯透明色之间的颜色过渡长度近似于模 糊半径的两倍(比如在这里是 8px)。

  • 单侧投影

    right

    box-shadow: 5px 0px 10px -5px blueviolet;

    left

    box-shadow: -5px 0px 10px -5px blueviolet;

    bottom

    box-shadow: 0px 5px 10px -5px blueviolet;

    top

    box-shadow: -5px 5px 10px -5px blueviolet;

  • 邻边投影

    right-top

    box-shadow: 5px -5px 10px -5px blueviolet;

    right-bottom

    box-shadow: 5px 5px 10px -5px blueviolet;

    left-top

    box-shadow: -5px -5px 10px -5px blueviolet;

    left-bottom

    box-shadow: 0px -5px 10px -5px blueviolet;

  • 双侧投影

    left-right

    box-shadow: 5px 0 5px -5px blueviolet, -5px 0 5px -5px blueviolet;

    top-bottom

    box-shadow: 0 5px 5px -5px blueviolet, 0 -5px 5px -5px blueviolet;
